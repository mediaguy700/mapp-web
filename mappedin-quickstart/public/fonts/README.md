# Grift Font

Add Grift font files here for the Pair Bracelet page:

- `Grift-Regular.woff2`
- `Grift-Medium.woff2`
- `Grift-SemiBold.woff2`
- `Grift-Bold.woff2`

Grift is a commercial font. Obtain it from [MyFonts](https://www.myfonts.com/collections/grift-font-38-lineart/) or your font provider, then export/convert to WOFF2 format.
